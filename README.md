## Red Dead Redemption 2 Easy Mod Installer
This Python code will make installing mods for Red Dead Redemption 2 a breeze. With just a few simple steps, you can have your favorite mods up and running in no time.

## Installation
1. Download the .py file
2. Open the file using a Python interpreter or IDE
3. Locate your Red Dead Redemption 2 folder (e.g., C:\Program Files (x86)\Steam\steamapps\common\Red Dead Redemption 2)
4. Enter the path to your RDR2 folder in the command prompt when prompted
5. Locate the mod folder you want to install and enter its path when prompted
## Alternative Installation
1. Locate your RDR2 folder
2. Simply copy the files from the mod folder into the main RDR2 directory.
